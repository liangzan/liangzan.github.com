Professional Work

I code for a living. Most of my professional work is done in Ruby, the Rails web framework, server side Javascript Node.js and Javascript MVC frameworks Backbone.js and Ember.js.

I have some devops experience. I have production experiences with deployment tools like Chef and Ansible. I'm familiar with cloud hosts like AWS, Rackspace and other VPS.

I've contributed to open source. Most of them can be found on my Github profile.

I'm also the author of SmugFTP and Notifymode(defunct).

SmugFTP is an uploader for Smugmug. It's written in Node.js and was previously hosted on Amazon AWS. I built it as I wasn't happy with the web uploaders that Smugmug provided.

Notifymode(defunct) does application monitoring for Node.js apps. It helps to profile Node.js applictions. You can use it to pin point code which are slow. I built it as I wanted a tool to help me profile SmugFTP and there are none available.

I also develop themes and plugins for Octopress at Octopress Themes.

I'm an independent consultant available for hire. If you are looking for a designer, look no further. My sister does that for a living.
